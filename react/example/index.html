Media Apron - React

媒体播放组件库示例 - Apron Design 分支项目

单个视频（带 WebVTT 字幕）

使用 src 属性指定视频源，cc 属性添加 WebVTT 格式字幕。点击设置按钮（⚙️）可以开关字幕。

单个视频（带 SRT 字幕）

使用 src 属性指定视频源，cc 属性添加 SRT 格式字幕。SRT 格式会自动转换为 WebVTT 格式。点击设置按钮（⚙️）可以开关字幕。

播放列表（简单模式）

使用 source 属性传入字符串数组

播放列表（完整模式 + CC 字幕）

使用 source 属性传入对象数组，包含标题、摘要、封面、字幕等信息。第一个视频带有字幕。

单个音频（无封面）

使用 src 属性指定音频源，无封面时控制区域居中显示播放按钮

单个音频（无封面 + 歌词）

使用 src 和 lyrics 属性，无封面但显示滚动歌词

单个音频（有封面 + 歌词）

使用 src、poster 和 lyrics 属性，显示封面图和同步滚动的 LRC 歌词

音频播放列表（简单模式）

使用 source 属性传入字符串数组，可以切换上一首和下一首

音频播放列表（完整模式 + 歌词）

使用 source 属性传入对象数组，包含标题、摘要、封面、歌词等信息。第一首歌曲带有歌词。
