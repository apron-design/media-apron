# Changelog

All notable changes to this project will be documented in this file.

## [26.0.0] - 2025-12-23

### Added

#### Video 组件
- ✨ 完整的视频播放功能
- 🎨 美观的自定义 UI 设计
- 📱 16:9 响应式布局，默认填满容器宽度
- 📋 支持播放列表（简单模式和完整模式）
- ⚡ 自动隐藏控制栏（3秒无操作）
- 🎯 沉浸模式和全屏模式
- 🔊 智能音量控制（根据音量级别显示不同图标）
- ⏩ 播放速度调节（0.25x - 2x）
- 📝 CC 字幕支持（透明背景，文字阴影）
- 🎬 封面图和视频信息显示
- ⌨️ 键盘快捷键（ESC 退出沉浸/全屏）
- 🎨 可自定义主题色
- ⏸️ 进度条预览和精确定位
- 🔄 Loading 加载动画
- 🎨 8px 圆角（默认模式），全屏无圆角

#### Audio 组件
- ✨ 完整的音频播放功能
- 🎨 简洁的 UI 设计
- 📐 无封面时 60px 高度，播放按钮居中
- 🖼️ 有封面时 16:9 比例，封面 contain 显示
- 📋 支持播放列表
- ⏮️ 上一首/下一首控制
- 🔊 音量控制
- ⏸️ 进度条和时间显示
- 🔄 Loading 加载动画
- 🎨 深灰色背景，8px 圆角

#### 通用特性
- 📦 TypeScript 完整类型支持
- 🎨 CSS Modules 样式隔离
- 🌐 跨浏览器兼容
- 📱 响应式设计
- ♿ 无障碍支持
- 🎨 矢量图标系统
- 🔧 高度可定制

### Technical
- React 16.8+ 支持
- TypeScript 5.0
- Vite 5.0 构建
- ESM 和 CJS 双模块格式
- Source Maps 支持
